Fizz Bot

League of Legends Discord bot which allows you to get updates when users enter or exit games, and access information you would normally have to access in a web browser right from your Discord server.

Track Players

Add users to your server's tracked list to get updates when players enter or exit games! See what champs are being played, gamemode, and more when players go into games.

Player Information

Eliminates need to open stats websites. Check player rank, match history (coming soon), and more with commands from your discord server.

More

There are more features and commands being worked on every week.

Commands

You can find the list of commands and their function here. The command prefix for the bot is '%'.

%help

DMs you the list of commands that are implemented. You can do %help [command name] for more information on a specific command.

%add

Adds a user to your server's list of tracked users. Currently, only NA players are supported, with multiregion support coming soon. %add [summoner name]

%clear

Clears the users that are being tracked for your server.

%ingame

Check who is playing right now.

%list

Sends the list of all tracked users for your server, along with the level

%opgg

Links to a player's op.gg. %opgg [summoner name]

%rank

Checks a player's rank. %rank [summoner name]

%remove

Removes a player from the tracked list. %remove [summoner name]

%setchannel

Sets the current channel to have player updates pushed to it.

%updateranks

Retrives and updates ranks for all users on the list (and plays a cute animation GIF).
